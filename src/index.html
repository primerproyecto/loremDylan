Some endpoints to recieved some dylan's song in ```json``` format. 

        
          fetch('https://loremdylan-production.up.railway.app/api/v1/all')
            .then(response => response.json())
            .then(json => console.log(json))

  • All dylan's songs

                
              fetch('https://loremdylan-production.up.railway.app/api/v1/all')
                .then(response => response.json())
                .then(data => console.log(data))

  • 10 songs from bobdylan

                
              fetch('https://loremdylan-production.up.railway.app/api/v1/10')
                .then(response => response.json())
                .then(data => console.log(data))

  • All dylan's song paginated

                
              fetch('https://loremdylan-production.up.railway.app/api/v1/10')
                .then(response => response.json())
                .then(data => console.log(data))
