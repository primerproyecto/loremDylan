While learning to webdeveloping, I missed some "old-school" content to diplay on the UI. I got tyred of rickymorty and pokeapi.

Don't get me wrong but those cartoons make no sense to me, I'am on my early fifties and I'll rather work with some other value content to display on the screen while learning how to work with api rest.

No bussiness (apart from bobdylan's bussiness) could be generated with these endpoints. Fell free to use these endpoints for learning purposes and give support to dylan.

In all returned data items, there's a link to the official bobdylan.com webpage that you could use to give extra infomation about the item as giving bob dylan all the credit that he deserves.

  • All dylan's songs

                
fetch('https://apidylan.up.railway.app/api/v1/all')
  .then(response => response.json())
  .then(data => console.log(data))

  • All dylan's song paginated

                
fetch('https://apidylan.up.railway.app/api/v1/paginated?page=2')
  .then(response => response.json())
  .then(data => console.log(data))

  • One random dylan's sentence

                
fetch('https://apidylan.up.railway.app/api/v1/sentence')
  .then(response => response.json())
  .then(data => console.log(data))
